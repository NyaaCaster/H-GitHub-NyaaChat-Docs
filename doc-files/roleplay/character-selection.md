# 角色选择界面

角色选择是 NyaaChat 最常使用的弹窗之一——在这里你可以浏览、切换、创建和管理所有角色。

## 打开角色选择

点击顶部工具栏第二行左侧的角色名按钮（✨ 图标 + 当前角色名），弹出 **"角色选择"** 窗口。

## 界面构成

弹窗标题为 **"角色选择"**（✨ 图标），内部从上到下分为：

- **标题栏右侧**：**"共享角色库"** 按钮（Library 图标）——点击可浏览社区共享的角色。
- **角色卡片列表**：每张卡片左侧为 64×96 角色封面缩略图，右侧显示角色名和描述（最多两行）。当前活跃角色以蓝色边框 + 蓝色勾选标记高亮。共享角色显示紫色 **"共享"** 标签。
- **底部操作区**：左侧 **"导入角色"**（Upload 图标），右侧 **"创建角色"**（Plus 图标）。
- **存储信息**：两张进度条——"角色卡储存"（默认 32 MB 上限）和"共享卡槽占用"（N / N 个）。

每张角色卡片悬停时，右上角会出现操作按钮：分享、编辑、删除。

## 浏览与搜索角色

角色列表为单列滚动，按添加顺序排列。目前**不支持搜索和排序**——角色按本地列表顺序展示，通过上下滚动浏览。如果你有大量角色，建议定期清理不再使用的卡片以保持列表整洁。

> 共享角色库中支持搜索、排序、标签筛选等高级功能。详见 [共享角色库](./shared-library)。

## 切换当前角色

点击任意角色卡片，该角色立即切换为当前活跃角色。NyaaChat 会自动为该角色新建一个对话（每个角色拥有独立的聊天历史）。

切换角色**不会删除**之前的对话——原角色的聊天记录保留在聊天记录弹窗中。

## 角色卡储存

角色卡储存显示你的本地存储使用情况：

- **标签**："角色卡储存"，默认上限 **32 MB**。
- 当已用空间超过上限的 95% 时，导入或创建角色会触发 **"储存空间不足"** 对话框，提示前往扩容。
- 点击对话框中的 **"前往扩容"** 按钮，会打开账号面板（需已登录），可花猫粮扩容角色卡储存（5 猫粮 / +12 MB）。

共享卡槽占用条显示你已获取的共享角色数量：

- **标签**："共享卡槽占用"（Wallet 图标），显示 `N / N 个`。
- 卡槽上限初始为 10。卡槽满时无法获取更多共享角色，需前往扩容（15 猫粮 / +5 卡槽，上限 200）。

> 关于扩容的详细操作，参见 [账号系统：额度与扩容](../account/quota-expand)。

## 新建与编辑入口

| 操作 | 入口 | 说明 |
|------|------|------|
| **创建角色** | 底部"创建角色"按钮 | 打开角色编辑弹窗（空白新建） |
| **编辑角色** | 卡片悬停 → 编辑图标 | 打开角色编辑弹窗（已有数据） |
| **分享角色** | 卡片悬停 → 分享图标 | 发布到共享角色库（需登录） |
| **删除角色** | 卡片悬停 → 删除图标 | 永久删除（需确认） |

> 创建角色的完整指南参见 [创建角色·基础](./create-character)。

## 删除角色

点击角色卡片上的 🗑️ 删除图标，弹出确认对话框：

- 普通角色：`确定要删除角色「角色名」吗？此操作不可撤销。`
- 共享角色：`确定要从本地删除共享角色「角色名」吗？删除后账号共享卡槽占用 -1，此操作不可撤销。`

点击 **"删除"** 按钮后角色将被永久移除。如果这是你唯一的角色，系统会阻止删除——至少需要保留一个角色。

共享角色被删除后，你可以在共享角色库中重新获取（只要原作者未下架）。

> 共享角色的更新：如果共享角色作者发布了新版本，卡片上的更新按钮（RefreshCw 图标）会出现琥珀色圆点标记。点击即可更新到最新版本，无需删除后重新获取。
