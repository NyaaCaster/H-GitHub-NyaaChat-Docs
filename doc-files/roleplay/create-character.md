# 创建角色·基础

创建角色是 NyaaChat 角色扮演的核心——一个精心编写的角色描述能让 AI 的回复风格截然不同。本文带你了解角色编辑器的每个字段和用法。

## 打开角色编辑弹窗

有两种方式打开角色编辑器：

1. **创建新角色**：角色选择弹窗 → 底部 **"创建角色"** 按钮。弹窗标题为 **"创建角色"**。
2. **编辑已有角色**：角色选择弹窗 → 悬停角色卡片 → 点击 ✏️ 编辑图标。弹窗标题为 **"编辑角色"**。

## 基础字段

角色编辑器包含五个基础字段，按从上到下的顺序排列：

### 角色封面

显示在角色选择卡片和聊天界面的角色缩略图上。

- 点击虚线框区域 → 选择一张图片 → 自动打开裁剪工具。
- 裁剪后会缩放为 **512×768** 像素。
- 图片存储在浏览器本地，不影响其他角色的储存空间。
- 已设置封面后，悬停会显示 X 按钮，点击可移除封面。

> 建议使用 PNG 或 WebP 格式，画质好且文件小。封面不是必须的——没有封面也不影响角色功能。

### 角色名

角色的名字，显示在顶栏左侧按钮、角色选择卡片和聊天界面上。`<input>` 文本输入框，示例 placeholder 为 `猫娘`。

**这是唯一必填字段**——角色名为空时无法保存。

### 角色概述

角色编辑器的**核心字段**。这里描述角色的性格、身份、说话方式和知识背景。这是 `textarea`（最低高度 120px），没有字数限制。

placeholder 提供了方向：`描述角色的核心设定、说话方式等...`

**编写建议**：

- 写清楚角色的身份和性格："你是一只来自异世界的猫娘，性格傲娇但内心善良。"
- 描述说话风格："用短句，句末偶尔加'喵'，对主人亲昵但对陌生人冷淡。"
- 加入背景知识："你知道中世纪的剑术和魔法体系，擅长草药学。"
- 避免矛盾设定——AI 会尝试遵循所有描述，互相矛盾的设定会导致回复不稳定。

> 💡 角色概述是**静态的**——它在整个对话中始终生效。如果你希望某些设定只在特定条件下出现（比如只有在提到某个词时才触发），需要用"角色规则（世界书）"来实现。参见 [角色规则·世界书](./world-info)（P4）。

### 第一条消息（开场白）

角色在对话开始时**自动发送**的第一条消息。这是可选的 `textarea`（最低高度 80px），不填的话角色会在收到你的第一条消息后才开始回复。

placeholder：`对话开始时角色自动发送的第一条消息（可选）...`

**编写建议**：

- 开场白应该体现角色的说话风格和当前场景。
- 如果你想让对话从一个特定情节开始，就在开场白中设定场景。
- 例如一只咖啡店猫娘的开场白：`"欢迎光临喵……啊，是主人啊！今天想喝点什么？本店特调猫爪拉花拿铁，只要 30 喵币喵~"`

### 规则条目（世界书）

"规则条目 (World Info)" 是一个列表区域，可以添加、编辑、删除**世界书规则**。

默认是空的。点击 **"添加规则"** 按钮可以创建一条新规则。每条规则包含关键词、触发方式、插入位置、约束强度等设置。

> 世界书规则是进阶功能，本文不做展开。完整指南见 [角色规则·世界书](./world-info)（P4）。

## 宏（变量占位符）

角色编辑器的界面**没有显式的宏按钮**，但你可以在角色概述、第一条消息等文本中直接使用以下变量。它们在提示词构建时会自动替换为实际值：

| 宏 | 展开为 | 使用场景 |
|----|--------|----------|
| <code v-pre>{{char}}</code> | 当前角色名 | 角色对话中引用自己的名字 |
| <code v-pre>{{user}}</code> | 当前用户名（或 "user"） | 角色对用户说话时引用 |
| <code v-pre>{{persona}}</code> | 当前用户人设描述 | 让角色了解用户的人设 |

此外，如果你创建了**角色变量**，也可以在文本中通过 <code v-pre>{{变量名}}</code> 引用（参见 [角色变量](./variables)（P4））。

> 这些宏和变量是在后端提示词构建时解析的——你在编辑器中看到的是原始文本，实际发送给 AI 时会替换为当前值。

## 保存角色

编辑完成后，点击弹窗底部右侧的 **"保存角色"** 按钮（Save 图标）。角色名称为空时按钮不可点击。

保存后角色会被写入浏览器的本地 IndexedDB 数据库。角色选择列表中会立即出现新角色或更新后的角色。

> 角色编辑器中未显示的字段（正则脚本、变量、扩展数据等）在保存时**不会被覆盖或丢失**——编辑器只管理它展示的那五个字段。
