# 用户角色·你的人设

NyaaChat 不仅让你为 AI 创建角色——你也可以为自己定义人设。用户角色告诉 AI"你是什么样的人"，让对话更真实、更沉浸。

## 什么是用户角色

用户角色（User Role）是**你在对话中扮演的人设**。它不是你创建来聊天的 AI 角色——它是你给自己定义的"身份卡"。

当你选择了一个用户角色后，NyaaChat 会在提示词中插入一段 `[User Persona:]` 标签，把你自己的人设描述告诉 AI。AI 会基于这个信息来调整它和你互动的方式。

例如：你创建了一个"异世界冒险者"的用户角色，描述为"来自远方的剑士，沉默寡言但可靠"。AI 角色在和你对话时就会默认你是这个身份，用符合冒险者语境的方式和你互动。

## 打开用户角色选择

点击顶部工具栏第二行的 👤 **用户角色** 按钮，弹出 **"用户角色选择"** 窗口。

窗口中列出你创建的所有用户人设。点击任意一条即可切换为当前人设。选中的项目会用蓝色边框高亮。

## 创建你的人设

在用户角色选择弹窗中，点击 **"创建用户角色"** 按钮，打开用户角色编辑器。

编辑器包含两个字段：

| 字段 | 输入类型 | placeholder | 说明 |
|------|---------|-------------|------|
| 用户名称 | `<input>` | `e.g. user` | 人设的简短名称，用于在列表中区分不同人设 |
| 用户人设描述 | `<textarea>`（120px 高） | `描述用户的人设，例如性格、背景、特殊偏好等...` | 核心字段。写你的性格、身份、说话方式、背景故事、与 AI 角色的关系等 |

示例人设描述：

> 你是一位来自北方雪国的冒险者，性格沉默寡言但内心善良。你擅长双手剑术，曾在王都守卫团服役。你对魔法一无所知但充满好奇。说话简短直接，不喜欢长篇大论。

两个字段都没有字数限制。填写完毕后点击 **"保存用户角色"** 按钮。

## 在对话中如何生效

选择用户角色后，人设以以下方式注入到对话中：

1. 系统提示词中插入 `[User Persona:]` 段落，后跟你的用户名称和人设描述。
2. <code v-pre>{{persona}}</code> 宏在角色文本中展开为你的人设描述——角色可以在开场白或世界书规则中引用这个宏来了解你。

一个角色可以搭配不同用户角色使用。比如同一个"冒险酒馆老板娘"AI 角色，用"新手冒险者"用户角色 vs "王国通缉犯"用户角色，对话氛围完全不同。

## 切换与管理人设

| 操作 | 方式 |
|------|------|
| **切换人设** | 用户角色选择弹窗 → 点击目标人设 → 自动生效 |
| **编辑人设** | 悬停列表项 → 编辑图标（如有） |
| **导出人设** | 编辑弹窗 → 底部左侧"用户角色导出"按钮 → 下载 JSON 文件（`NyaaChatUser-时间戳.json`） |
| **删除人设** | 列表项删除操作（如有） |

> 用户角色导出为**纯 JSON 文件**（非 PNG 角色卡格式），内容仅为 `{ name, profile }`。导入功能暂未提供——可通过导入全局设置文件恢复。
