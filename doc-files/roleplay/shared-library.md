# 共享角色库

共享角色库是 NyaaChat 的社区角色分享平台。在这里你可以浏览其他玩家发布的角色，一键获取使用，也可以把自己精心调试的角色分享给社区。

## 什么是共享角色库

一个在线的角色卡仓库——玩家们把自己创建的角色上传到这里，其他用户可以浏览、搜索、获取并使用。所有共享角色按作者和标签分类，支持搜索和排序。

浏览和获取共享角色**不需要登录**，但如果想发布自己的角色、或者给别人的角色点赞/差评，则需要登录 NyaaAcount。

## 打开共享库

1. 点击顶部工具栏第二行左侧的角色名按钮 → 打开 **角色选择** 弹窗。
2. 点击角色选择标题栏右侧的 📚 **"共享角色库"** 按钮（Library 图标）。
3. 弹出 **"共享角色库"** 窗口。

## 浏览与获取共享角色

### PC 端布局

- **左侧面板**：搜索框 + 排序下拉 + 标签列表（支持按标签筛选）
- **右侧内容区**：3 列网格，每张卡片显示封面、角色名、作者、简介、标签、统计数据

### 移动端布局

- 顶部搜索栏 + 排序按钮 + "标签"按钮（弹出底部弹窗选择标签）
- 内容区为单列列表

### 搜索与筛选

| 功能 | 说明 |
|------|------|
| **搜索** | 输入关键词，搜索作者名、角色名、简介、标签。输入后 400ms 自动触发搜索。 |
| **排序** | 四个选项：更新时间、下载数、好评数、差评数。点击切换升序/降序。 |
| **标签筛选** | 左侧标签列表，点击筛选。默认"全部"。 |
| **作者筛选** | 点击卡片上的 `@作者名` 即按该作者筛选。有筛选时显示"取消"按钮。 |

空结果时显示："没有符合条件的共享角色"。

### 卡片信息

每张共享角色卡片展示：

- **封面**：2:3 比例缩略图，无封面时显示占位图标
- **原创/转载** 标签
- **角色名** + **作者名**（点击作者名可按该作者筛选）
- **更新时间**
- **简介**（最多 2 行截断）
- **标签**（最多 4 个）
- **下载数** / **好评数** / **差评数**

作者自己的卡片右上角额外显示 ✏️ 编辑和 🗑️ 删除按钮。

### 获取角色

点击卡片上的绿色 **"使用"** 按钮（CloudDownload 图标）即可获取。

获取流程：

1. 系统检查你的**共享卡槽**是否已满。
2. 卡槽未满 → 角色以共享类型加入本地角色列表 → 自动切换为该角色。
3. 卡槽已满 → 弹出 **"共享卡槽已满"** 对话框：

   > 当前共享卡槽已达上限（N 个），无法继续获取共享角色。
   > 请清理不再使用的共享角色，或前往扩容（15 猫粮 / +5 卡槽）。

   - 点击 **"前往扩容"** → 打开账号面板进行扩容。
   - 点击 **"取消"** → 关闭提示，不获取。

> 共享卡槽上限初始为 10。详见 [账号系统：额度与扩容](../account/quota-expand)。

获取不需要登录，也不需要消耗猫粮——**完全免费**。但如果服务器返回错误，可能是角色已被删除或网络不可用。

## 评价与互动

每张卡片底部显示好评（👍）和差评（👎）按钮，你可以给喜欢的角色好评、给质量不佳的角色差评。

- **需要登录**。未登录时点击评价按钮会弹出登录窗口。
- 再次点击已选的评价方向可以**取消**评价。
- 切换到另一个评价方向会自动翻转。

评价数是一种社区反馈——好评多的角色在排序中会靠前。

## 分享你的角色

将你本地创建的角色发布到共享角色库：

1. 打开 **角色选择** 弹窗。
2. 悬停你要分享的角色卡片 → 点击右上角 ☁️ **分享角色** 图标。
3. 弹出一个警告确认框：
   > 您将为自己公开分享的角色承担所有责任，请三思而后行。
   > ⚠ 请勿转载分享「类脑」和「旅途」作者发布的角色卡。

4. 点击 **"同意"**。
5. 系统检查登录状态（未登录弹出登录窗）。
6. 打开角色分享编辑界面，填写来源（source）、简介（intro）、标签（tags）等信息。
7. 确认发布。

发布后，角色会出现在共享角色库中，其他用户可以浏览和获取。如果你后续修改了角色，可以重新发布更新——已在共享库的卡片上的"更新"按钮会提示有新版本。

> 编辑共享角色信息（来源/简介/标签）不会影响本地角色数据。

## 卡槽与管理

- **共享卡槽**是获取共享角色的"槽位"。初始上限为 **10 个**。
- 删除本地已获取的共享角色 → 卡槽占用 **-1**。
- 卡槽满 → 需扩容：**15 猫粮 / +5 卡槽**，上限 **200 个**。
- 作者删除自己发布的共享角色后，已获取该角色的用户**不受影响**（但无法再获取更新）。

## 登录要求与归属说明

| 操作 | 需要登录？ | 备注 |
|------|:--:|------|
| 浏览列表、搜索、排序、标签筛选 | ❌ | |
| 获取（使用）角色 | ❌ | 免费，但受卡槽上限限制 |
| 好评 / 差评 | ✅ | 未登录时弹出登录窗口 |
| 发布角色（分享） | ✅ | |
| 编辑自己发布的信息 | ✅ | |
| 删除自己发布的角色 | ✅ | Token 过期时弹出登录窗 |

### Token 过期处理

如果在评价或管理自己发布的角色时遇到 `unauthorized` 错误（401），NyaaChat 会自动清除本地登录状态并弹出登录窗口——重新登录即可。
