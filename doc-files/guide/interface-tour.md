# 界面导览

这篇文章带你认识 NyaaChat 主界面的每一个部分。它是后续所有板块的**通用术语基础**——你在这里看到的按钮名称、弹窗名称，会在整份文档中反复使用。

## 整体布局

NyaaChat 主界面采用经典的三段式布局：

```
┌─────────────────────────────────┐
│     顶部工具栏（两行）           │  ← 所有功能的入口
├─────────────────────────────────┤
│                                 │
│     聊天区                       │  ← 消息在这里展示
│                                 │
│                                 │
├─────────────────────────────────┤
│     输入区                       │  ← 打字、发送、模型选择
└─────────────────────────────────┘
```

NyaaChat 是一个**单页应用**——你看到的所有功能界面（角色管理、设置、知识库等）都是通过顶部工具栏的图标按钮唤起的**弹窗**，而不是跳转到独立的页面。弹窗可以拖拽、关闭，多个弹窗之间可以层叠。

## 顶部工具栏（第一行）

第一行工具栏位于界面最顶部，包含应用标题和全局功能按钮。

| 图标（从左到右） | 按钮名称 | 功能说明 |
|------------------|----------|----------|
| （应用标志） | NyaaChat 标题 | 显示应用名称和版本号，点击版本号可查看版本信息 |
| 🔲 | **全屏模式** | 切换浏览器全屏显示，再次点击退出全屏 |
| `>_` | **控制台** | 显示终端输出日志，用于排查问题或查看系统信息 |
| 🔥 | **绕过机制（Bypass）** | 管理模型的审查绕过策略，进阶功能 |
| ⚙️ | **设置** | 最常用的全局按钮——在这里配置对话模型、生图模型、发送模式等 |

> 此外还有一个 **扩展** 按钮（拼图图标），目前暂未对外开放，日常使用中你不需要关注它。

## 顶部工具栏（第二行）

第二行工具栏紧接在第一行下方，是所有角色和对话相关功能的入口。

| 图标/显示 | 按钮名称 | 功能说明 | 唤起弹窗 |
|-----------|----------|----------|----------|
| ✨ + 角色名 | **角色管理** | 浏览、切换、创建、导入角色 | 角色选择 |
| 🪪 | **用户账号** | 登录/注册、查看猫粮余额、扩容、云同步 | 用户账号 |
| 👤 | **用户角色** | 创建和管理你的人设（在对话中扮演的角色） | 用户角色选择 |
| 📖 | **知识库** | 创建和管理 RAG 知识库，上传文档 | 知识库管理 |
| `.*` | **正则** | 创建和管理正则规则，自动处理文本 | 正则 |
| 📋 | **聊天记录** | 查看、搜索、导出历史聊天记录 | 聊天记录 |
| ＋ | **新的对话** | 清空当前聊天，开始一个新对话 | （无弹窗，直接清空） |

> **注意**：部分按钮在未登录时不会显示或功能受限。登录后知识库、共享角色库等功能才会完整可用。

## 聊天区

聊天区占据了界面中央的主体空间。

**空状态（首次使用）**：当你还没有发送任何消息时，聊天区中央会显示"欢迎使用 NyaaChat"的提示，指引你去设置 API Key 和选择角色。

**消息气泡**：每次对话由两方消息交替构成：

- **你的消息**（用户消息）：橙色边框气泡，显示在右侧。
- **AI 的回复**（助手消息）：蓝色边框气泡，显示在左侧。每条 AI 消息下方有一行操作按钮。

**消息操作按钮**（出现在每条 AI 消息下方）：

| 图标 | 操作 | 说明 |
|------|------|------|
| 🔄 | **重新生成** | 用同一条用户消息重新请求 AI 回复 |
| 📋 | **复制文本** | 复制该消息的全部文本（已处理占位符变量） |
| ✏️ | **编辑** | 编辑该条消息的内容（仅用户消息支持） |
| 🗑️ | **删除** | 删除该消息（会弹出确认提示） |
| 🖼️ | **生成图片** | 基于消息内容生成图片（需已配置生图模型） |

## 输入区

输入区是界面底部的固定栏，也是你与 AI 交互的主要入口。

**文本输入框**：输入你想说的话。按 **Enter** 键发送（按住 Shift + Enter 换行）。可以在设置中切换为 Ctrl + Enter 发送。

**输入区工具栏**（输入框上方一行小图标）：

| 图标 | 工具 | 说明 |
|------|------|------|
| 📎 | **附件** | 添加图片等附件到消息中（需要模型支持多模态） |
| 🌐 | **联网搜索** | 开启后 AI 会联网搜索相关信息来增强回答 |
| 🔌 | **MCP 工具** | 接入外部工具扩展 AI 的能力 |
| （卡片） | **模型选择** | 快速切换当前对话使用的模型 |

**发送 / 停止按钮**：

- 空闲状态：蓝色 **发送** 按钮，输入框为空且无附件时不可点击。
- 生成中状态：红色 **停止** 按钮，点击可中断 AI 的回复生成。

输入框最底部有一行灰色小字："内容由 LLM 服务提供。生成内容仅供参考。"

## 弹窗交互约定

NyaaChat 中几乎所有功能都通过弹窗（Modal）来实现。了解弹窗的交互方式能帮你更高效地使用：

- **唤起**：点击顶部工具栏的对应图标按钮，弹窗从屏幕中央弹出。
- **关闭**：点击弹窗右上角的 X 按钮，或点击弹窗外的遮罩区域。
- **拖拽**：按住弹窗标题栏可以拖动弹窗位置。
- **层叠**：多个弹窗可以同时打开、层叠显示。例如在"设置"弹窗中点击"对话模型供应商"，会再弹出一个子级弹窗。
- **独立关闭**：关闭子级弹窗不会关闭父级弹窗。

弹窗内的具体操作将在各自功能板块中详细介绍。
