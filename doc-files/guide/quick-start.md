# 快速开始

在浏览器打开 NyaaChat，三步完成从零到第一次对话。

## 打开 NyaaChat

访问 **[http://h.nyaa.host:3095](http://h.nyaa.host:3095)**。

> **提示**：如果你之前使用过 NyaaChat 旧版，页面打开后可能会弹出一个"本地数据迁移"对话框——这是因为你的旧数据需要从旧版存储（上限 10 MB）迁移到新版 IndexedDB 存储（上限大幅提升）。点击 **开始迁移** 按钮，几秒钟即可完成，数据不会丢失。迁移完成后点击 **开始使用** 进入主界面。

首次打开时，你会看到一个空白的聊天页面，中间显示"欢迎使用 NyaaChat"的提示。接下来三步带你完成首次对话。

## 三步上手

### 第一步：配置对话模型

NyaaChat 不自带大模型，你需要先告诉它用哪个 API 来驱动对话。

1. 点击顶部工具栏第一行右侧的 **设置** 按钮（齿轮图标）。
2. 在设置弹窗中找到 **对话模型供应商** 一栏。
3. 在供应商列表中选择 **QinyAPI**（推荐新用户使用——注册简单、国内直连、性价比高）。
4. 选择一个接入点（推荐 `.icu`），然后填入你的 QinyAPI 密钥。
5. 点击 **管理模型**，勾选你想用的对话模型。

详细的配置步骤和截图参见 [配置对话模型](./configure-model)。

### 第二步：选择一个角色

NyaaChat 已经为你准备了一个默认角色——**猫娘**。你可以直接用她开始对话，也可以切换到其他角色。

1. 点击顶部工具栏第二行左侧的角色名按钮（显示当前角色名，旁边有一个 ✨ 图标）。
2. 在弹出的 **角色选择** 窗口中，点击任意角色卡片即可切换。
3. 第一次使用可以先不换——默认的"猫娘"就是一个不错的起点。

角色选择更多细节参见 [角色选择界面](../roleplay/character-selection)。

### 第三步：发出第一条消息

模型配好了，角色也选好了——现在可以聊天了。

1. 在底部的输入框中输入你的第一条消息（比如"你好！"）。
2. 按 **Enter** 键（或点击蓝色的 **发送** 按钮）。
3. 等待 AI 回复——发送按钮会变成红色的停止按钮，你随时可以点击它中断生成。

消息的更多玩法（重新生成、编辑、删除、复制）参见 [开始第一次对话](./first-chat)。

🎉 恭喜！你已经完成了首次对话。

## 遇到问题怎么办

| 问题 | 解决方法 |
|------|----------|
| API Key 无效/401 错误 | 检查密钥是否正确填入，详见 [配置对话模型：常见问题](./configure-model#常见配置问题) |
| 模型列表拉取失败 | 检查接入点选择是否正确，或尝试另一个接入点 |
| 不知道该选哪个模型 | QinyAPI 的 `gemini-2.5-pro` 和 `gpt-4o` 都是不错的选择 |
| 其他问题 | 查看 [常见问题](../faq) |
