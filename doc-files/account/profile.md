# 账号面板

登录 NyaaAcount 后，账号面板（UserAccountModal）是你的个人信息与控制中心。在这里你可以查看账号信息、管理扩容、了解共享统计。

## 打开账号面板

点击顶部工具栏第二行的 **用户账号** 按钮（🪪 IdCard 图标）。

- 如果尚未登录，会显示登录表单（用户名或邮箱 + 密码）。
- 登录后，会显示完整的账号信息面板。

## 基本信息

登录后的面板顶部是你的身份信息区域：

### 账号与用户名

| 字段 | 说明 |
|------|------|
| **账号** | 注册时填写的唯一账号 ID。以等宽字体显示，**不可修改**。这就是你的 NyaaAcount 账号名。 |
| **用户名** | 你的显示名称。点击旁边的 ✏️ 铅笔图标可以**内联编辑**——直接在原位置输入新用户名，最长 24 个字符，修改后即时保存。用户名会在共享角色库中公开展示。 |

### 注册时间

`YY-MM-DD hh:mm` 格式显示你的 NyaaAcount 注册时间。来自 NyaaAcount 服务端，不可修改。

### 猫粮余额

橙色 🥫 猫罐头图标 + 当前余额数字。右侧有 **"充值"** 按钮——点击会在新窗口打开 NyaaAcount 充值页面（`h.nyaa.host:5110/?view=recharge`）。

猫粮是 NyaaChat 的虚拟货币，用于扩容各类功能上限。详见 [猫粮与充值](./catfood-recharge)。

### 历史消耗

红色 📉 图标 + 累计猫粮消费总额。这是你从注册以来在扩容上花费的猫粮总计。

## 共享统计

"共享统计"区域以四格网格展示你在共享角色库的贡献情况：

| 统计项 | 含义 |
|--------|------|
| 共享角色数 | 你在共享角色库中发布过的角色总数 |
| 总下载量 | 你的所有共享角色被其他用户获取的总次数 |
| 好评数 | 你的共享角色收到的 👍 评价数 |
| 差评数 | 你的共享角色收到的 👎 评价数 |

> 💡 共享统计反映了社区对你发布的角色的认可程度。关于共享角色库的更多信息参见 [角色扮演：共享角色库](../roleplay/shared-library)。

## 面板上的常用操作

在账号面板中，你可以进行以下操作：

| 操作 | 位置 | 说明 |
|------|------|------|
| **扩容** | 经济数据区，各项旁"扩容"按钮 | 花猫粮增加共享卡槽/知识库栈/储存空间/图包上限。详见 [额度与扩容](./quota-expand) |
| **充值** | 猫粮余额旁"充值"按钮 | 跳转 NyaaAcount 充值页 |
| **账号管理** | 面板下方"账号管理"按钮 | 跳转 NyaaAcount 主页（`h.nyaa.host:5110`），管理账号详细设置 |
| **退出登录** | 面板底部红色"退出登录"按钮 | 退出当前登录，回到未登录状态 |
