# 账号系统总览

NyaaChat 的账号系统是**可选的补充功能**——不登录也能完整使用全部本地功能。登录 NyaaAcount 后，你会解锁跨设备同步、共享社区互动和功能上限扩容。

## 账号能做什么

| 功能 | 不登录（离线） | 登录 NyaaAcount |
|------|:--:|:--:|
| 创建/编辑/导入角色 | ✅ | ✅ |
| 对话、知识库、生图 | ✅ | ✅ |
| 浏览共享角色库 | ✅ | ✅ |
| 获取（下载）共享角色 | ✅ | ✅ |
| 评价共享角色（好评/差评） | ❌ | ✅ |
| 发布/编辑/删除自己共享的角色 | ❌ | ✅ |
| 云同步（设置 + 角色封面 + 聊天记录） | ❌ | ✅ |
| 扩容（卡槽 / 知识库栈 / 储存空间 / 图包） | ❌ | ✅ |
| 跨设备使用同一账号 | ❌ | ✅ |

## 打开账号面板

点击顶部工具栏第二行右侧的 **用户账号** 按钮（🪪 IdCard 图标）。

- **未登录**：显示登录表单——输入 NyaaAcount 账号密码即可登录。
- **已登录**：显示完整账号面板——猫粮余额、扩容选项、共享统计、账号管理。

## 本地使用与登录使用的区别

**不登录时**：你的所有数据（角色、聊天记录、设置、API Key、知识库等）全部保存在浏览器的本地 IndexedDB 数据库中。功能完全可用，但无法使用任何依赖云端服务的联网功能。这是 NyaaChat 的默认状态——隐私友好、零依赖。

**登录后**：登录态通过 Token 持久化存储在浏览器本地。你可以：
- 在设置中上传/下载云同步存档
- 在共享角色库中评价和发布角色
- 用猫粮扩容各种功能上限
- 多设备使用同一 NyaaAcount 账号，通过云同步接续资料

登录**不会**把你的本地数据自动上传到云端——云同步需要你主动操作（点击"上传设置"/"上传记录"等按钮）。

## 核心概念一览

### 猫粮

猫粮（CatFood）是 NyaaChat 的虚拟货币，图标为 🥫 猫罐头。它只用于**扩容**各类功能上限——不用于对话、不生图。猫粮余额来自 NyaaAcount 账户，充值在 NyaaAcount 站点完成。

> 详见 [猫粮与充值](./catfood-recharge)

### 额度与扩容

每种需要登录的联网功能都有初始上限（如共享卡槽初始 5 个、知识库栈初始 3 个）。达到上限后，你可以花猫粮扩容。扩容立即生效，**这是猫粮的唯一消费途径**。

> 详见 [额度与扩容](./quota-expand)

### 云同步

登录后，你可以将设置（含 API Key、供应商配置、角色元数据等）和聊天记录上传到云端，在另一台设备上下载恢复。聊天记录使用 `Nyaa-HMAC-XOR-V1` 算法端到端加密，只有你能解密。导入/导出（纯本地操作）也支持在无网络环境下备份。

> 详见 [云同步](./cloud-sync)
